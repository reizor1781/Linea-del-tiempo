dns

1984

En 1984 se creó el Sistema de Nombres de Dominio (DNS - Domain Name System), un sistema diseñado para facilitar la identificación de computadoras conectadas a Internet mediante nombres fáciles de recordar en lugar de direcciones numéricas.

Antes del DNS, para acceder a un servidor era necesario conocer su dirección IP (por ejemplo, 192.168.1.1), lo cual era complicado de memorizar y poco práctico a medida que la red crecía.

El DNS fue desarrollado por Paul Mockapetris y permitió traducir los nombres de dominio en direcciones IP de manera automática, haciendo que los usuarios pudieran acceder a sitios web escribiendo nombres como ejemplo.com en lugar de una secuencia de números.

Ese mismo año se introdujeron los primeros dominios de nivel superior (TLD), entre ellos: .com (comercial), .org (organizaciones), .edu (educación), .gov (gobierno) y .mil (militar).

Gracias al DNS, Internet pudo expandirse de manera organizada y escalable, convirtiéndose en una herramienta accesible para millones de usuarios en todo el mundo.
